Assignment 1

This project involved designing and assembling a laser-cut spherical model using MDF (prototyped in cardboard). The structure features an inner hollow sphere that is exactly 2/3 the size of the outer sphere. Designed for tool-free assembly, the model uses interlocking joints without glue, demonstrating precision in digital fabrication and 3D form construction.
